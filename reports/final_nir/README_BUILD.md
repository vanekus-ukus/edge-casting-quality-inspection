# Сборка НИР

Требуется XeLaTeX.

```bash
cd reports/final_nir
./build.sh
```

Основной файл: `main.tex`.

Итоговый PDF: `final_nir.pdf`.

Оформление:
- A4;
- шрифт Times New Roman;
- кегль 12;
- межстрочный интервал 1.5;
- поля: верх 2 см, низ 2 см, левое 3 см, правое 1.5 см;
- абзацный отступ 1.25 см;
- нумерация страниц снизу по центру.

На этой машине `Times New Roman` найден через fontconfig. Если на другой машине шрифт недоступен, в `main.tex` можно временно заменить его на `TeX Gyre Termes`, но для финальной подачи нужно проверить соответствие требованиям конкурса.

Перед подачей вручную заменить плейсхолдеры на титульном листе:
- `[образовательная организация]`;
- `[ФИО]`;
- `[научный руководитель при наличии]`;
- `[город]`.
